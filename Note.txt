Deployment using render

Make sure you have pushed your code to github.
go to render.com click dashboard and sign up with github.
After sign in click on add new and choose web service.
then select the repository you wannt to deploy which in our case is shopit and click connect.
then give it a name.

set your build command to npm run build.
set the start command to npm start
select free package

register your environment variable like MONGO_URI AND JWT_SECRET.
